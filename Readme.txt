Formulas utilizadas para crear nuevos datos:

score critico: =ROUND((RAND()*1+RAND()*1+RAND()*1+RAND()*1+RAND()*1+RAND()*8)-0.5,0)

score usuario : =ROUND((RAND()*2+RAND()*2+RAND()*2+RAND()*2+RAND()*2+1+1+1+1+2)-3.1,0)

global sales= US_Sales + EU_Sales + JP_Sales 

US_Sales: =RANDBETWEEN(1000000,1500000)
EU_Sales: =RANDBETWEEN(900000,1200000)
JP_Sales: =RANDBETWEEN(1400000,1800000)
